# GitHub Repository Dashboard

A lightweight Streamlit dashboard that summarizes repository health and activity at a glance.
It helps you scan your repos quickly, spot what changed recently, and drill into details when needed.

## Features
- [x] Repository overview table with key metadata and filtering.
- [x] Commit activity visualizations (trend line + heatmap).
- [x] Per-repository details panel with drill-down views.
- [x] Cache controls and refresh indicators for data sections.
- [ ] Features checklist and per-repo feature progress tracking. (in progress)
